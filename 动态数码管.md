# 动态数码管

### 需求
1. 用户输入0~9任意一个数值
2. 输出数码管形式的数字（详情见后面）
3. 数码管长宽比：4x5
4. 用户输入"+"、"-"可按倍数放大显示的数字,且最小为4x5，最大无上限（4x5、8x10、12x15....）
